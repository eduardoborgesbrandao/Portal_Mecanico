# Aplicativo Mobile SpeedZone

O app desenvolvido para SpeedZone é um aplicativo para a organização interna da empresa. Separando os projetos por status e implementando níveis de acesso para o gerentes e os mecânicos.
O aplicativo mobile da Speed Zone foi desenvolvido usando a tecnologia Flutter, um framework moderno da Google voltado para a criação de aplicações nativas, dinâmicas e de alta performance a partir de uma única base de código.
A arquitetura Baseada em Componentes foi adotada para organizar melhor o código, separando as responsabilidades entre as Classes/Modelos (estruturas de dados), os Widgets (interface de visualização) e a Lógica de Estado (controlando o comportamento das telas).
Isso facilita a manutenção, reutilização de componentes (Widgets) e a rápida evolução do sistema em diversas plataformas. Veja a documentação completa do aplicativo neste [link.](https://github.com/eduardoborgesbrandao/Portal_Mecanico/wiki)
